# 表格
## 无序表格
```html
<!-- 表格前面为小方块 -->
<ul type="square">
  <li>Milk</li>
  <li>Toilet Paper</li>
  <li>Cereal</li>
  <li>Bread</li>
</ul>
<!-- 表格前面为小圆点 -->
<ul type="disc">
  <li>Milk</li>
  <li>Toilet Paper</li>
  <li>Cereal</li>
  <li>Bread</li>
</ul>
<!-- 表格前面为空心原圈 -->
<ul type="circle">
  <li>Milk</li>
  <li>Toilet Paper</li>
  <li>Cereal</li>
  <li>Bread</li>
</ul>
```
